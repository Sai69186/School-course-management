GitHub Deployment Commands
========================

After creating your repository on GitHub, run these commands:

1. Add remote repository (replace YOUR_USERNAME with your GitHub username):
   git remote add origin https://github.com/YOUR_USERNAME/edumanage-student-management.git

2. Push to GitHub:
   git push -u origin main

3. Verify deployment:
   git remote -v

Alternative repository names if the above is taken:
- edumanage-system
- student-management-react
- edumanage-platform
- school-management-system
